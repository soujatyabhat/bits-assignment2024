Web Servers


  • Hardware and Software: A web server can refer to either a computer (hardware) that stores web server software and a websiteâ€™s component files, or the software that serves these files over the internet.

  • HTTP Protocol: Web servers operate on the Hypertext Transfer Protocol (HTTP), which is the foundation of data communication on the World Wide Web.

  • File Delivery: When a browser requests a file hosted on a web server, the server finds the requested document and sends it back to the browser through HTTP.

  • Static and Dynamic: There are static web servers that send hosted files as-is to your browser, and dynamic web servers that update the hosted files before sending content to your browser via the HTTP server.

  • High Availability: In a multi-tier web server architecture, multiple servers are used to distribute the workload and ensure high availability. This often involves load balancers that evenly distribute incoming requests across a cluster of web servers.
